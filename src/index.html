功能介绍

抖音直播录制工具

版本 1.0.0

一款基于 Electron 的 Windows 桌面应用，可自动录制抖音直播间画面（含弹幕和礼物特效），支持多直播间同时监控与录制。

核心功能

  • 🔗
    多种添加方式

    支持房间号、主播主页链接、直播间链接、分享文本等多种方式添加直播间

  • 📡
    自动监控与录制

    定时检测直播状态，开播自动开始录制，无需手动操作

  • 🎬
    完整画面捕获

    通过离屏渲染技术，录制完整的直播画面，包括视频内容、弹幕和礼物特效

  • 📁
    按主播分类存储

    录制文件按主播名称自动创建子文件夹，文件名包含时间戳，方便查找管理

  • 📋
    录制记录

    记录每次录制的开始/结束时间、文件大小等信息，支持打开文件所在目录

  • ⚙️
    灵活配置

    支持自定义输出路径、检测间隔、帧率等参数，可设置开机自启动

使用说明

  1. 点击"登录抖音"按钮登录你的抖音账号
  2. 在上方输入框中粘贴房间号、主播主页链接或直播间链接
  3. 点击"添加"按钮将直播间添加到监控列表
  4. 开启"自动录制"后，检测到开播将自动开始录制
  5. 也可点击"开始录制"按钮手动控制录制
