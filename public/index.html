使用说明

  1. 解压下载的压缩包到任意目录
  2. 双击 一键封装.bat 运行
  3. 脚本自动检测环境，缺失的 Node.js / pnpm 会自动下载安装
  4. 安装完成后自动打包生成 EXE 安装包
  5. 生成的 EXE 在 dist/ 目录中
注意：首次封装需要联网下载依赖（约 500MB），请确保网络畅通。 封装过程日志保存在 build_log.txt 中，如遇问题可据此排查。
